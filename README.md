# Housing-Market-Analysis-Dashboard-Power-BI-

This repository contains an interactive Microsoft Power BI dashboard designed to analyze housing market trends, sales performance, and property characteristics across regions and house types.
The dashboard is structured into three main pages, each focusing on a different analytical perspective.
